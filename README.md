## About

## Setup

1. Clone repository

```bash
git clone https://github.com/denkisil/denkisil-site
```

2. Go to folder and install depedencies

```bash
npm i
# or
yarn
```

3. For development server run

```bash
npm run start
```

4. For production build run

```
npm run build
```
